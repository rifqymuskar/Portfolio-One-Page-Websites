# Installation
 1. Create Database from mySQL with name resume
 2. Import Database from main folder /dist/sql/resume.sql
 3. start xampp
 4. open link browser localhost/resume

 1. Backend : Codeigniter + AJAX
 2. Frontend : jQuery + Materialize CSS
 3. responsive : 80%
